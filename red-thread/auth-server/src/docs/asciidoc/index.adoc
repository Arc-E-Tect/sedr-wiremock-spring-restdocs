= Auth-Server API Documentation

Welcome to the Auth-Server API documentation.

This documentation provides an overview of the Auth-Server APIs, including how to use them, how to register as a developer, and how to request an API Key.

In case you are looking for the API reference documentation, you can find it in the https://developer.red-thread.com/reference[API Reference] section of the Red Thread developer portal.

TIP: Any questions or issues related to the API can be reported in the https://developer.red-thread.com/issues[Red Thread API Issue Tracker] on the Red Thread developer portal.
